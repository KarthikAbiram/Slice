# Slice
Slice is a tool for merging and slicing (row wise and column wise filtering) of *.csv files.

# Status
Check if existing tools can satisfy the above needs before trying to create one.

# Existing Tools
- [csvkit](https://github.com/wireservice/csvkit) - 6k stars, Python
- [qsv](https://github.com/dathere/qsv) - 3k stars, Rust
- [xan](https://github.com/medialab/xan) - 3k stars, Rust
- [xsv](https://github.com/BurntSushi/xsv) - 10k stars, Rust, now unmaintained.
